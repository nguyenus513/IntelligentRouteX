# IntelligentRouteX

## One-click demo

```powershell
.\scripts\irx.ps1 up
.\scripts\irx.ps1 status
.\scripts\irx.ps1 test -Quick
.\scripts\irx.ps1 down
.\scripts\irx.ps1 package
```

- Playground: `http://localhost:5173/playground`
