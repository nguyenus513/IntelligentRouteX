# Dispatch V2 Quality Metrics

selectedProposalCount is the number of proposals selected for execution.

Quality benchmark metrics include selectedProposalCount, executedAssignmentCount, conflictFreeAssignments, bundleRate, routeCostQuality, driverEntryQuality, dispatchRegretAverage, courierUtilizationEstimate, and cancel-risk proxy.

Ablation modes include local-real, tabular on/off, and OR-Tools vs degraded greedy.
